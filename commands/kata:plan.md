# Kata Plan

Run the Kata planning workflow.

## What To Do

1. Read `skills/kata/workflows/planning.md` if available.
2. Use installed planning skills first, such as `superpowers:brainstorming` and `superpowers:writing-plans` when available.
3. Brainstorm and clarify the request into `docs/spec.md`.
4. Only after the spec is decision-ready, update `docs/prompt_plan.md` and `docs/todo.md`.

## Equivalent CLI

```bash
kata plan
```
